# 🚀 InterviewGen AI

AI-powered interview preparation platform that analyzes a candidate's resume, job description, and self-description to generate personalized interview reports using Google Gemini AI.

## Features

- Secure Authentication
- Resume Upload (PDF)
- AI Interview Report Generation
- Match Score
- Technical Questions
- Behavioral Questions
- Skill Gap Analysis
- Preparation Roadmap
- Report History
- Admin Dashboard
- Analytics
- User Management
- Report Management
